# Интерактивное моделирование радиоактивного распада

Образовательный проект для изучения физических процессов.

## Как запустить

Просто откройте файл `index.html` в браузере.

## Разделы

### 1. Радиоактивный распад
- **Формула**: N(t) = N₀ e^(-λt)
- **Параметры**:
  - N₀: начальное количество ядер (100-10000)
  - λ: постоянная распада (10⁻⁴ до 10⁻¹ 1/с)
  - Время моделирования автоматически подстраивается под период полураспада
- **Графики**: количество ядер N(t) и активность A(t) = λN(t)

### 2. Цепная реакция (непрерывная модель)
- **Формула**: N(t) = N₀ e^(αt), где α = ln(k)/τ
- **Параметры**:
  - k: коэффициент размножения (0.5-2)
  - τ: время одного поколения (0.1-5 мкс)
  - N₀: начальное количество нейтронов

### 3. Дискретная модель (по поколениям)
- **Формула**: Nₙ = N₀ · k^n
- **Столбчатая диаграмма** показывает рост/затухание по поколениям

### 4. Энергетический выход
- **Формула**: E_tot(t) = E_один · N₀(1 - e^(-λt))
- Расчет выделенной энергии и средней мощности

### 5. Сравнение процессов
- Нормализованное сравнение распада и цепной реакции

## Технологии

- HTML5 + CSS3 + JavaScript
- Chart.js для графиков
- MathJax для отображения формул

## Диапазоны значений

**Постоянная распада λ:**
- 10⁻⁴ (T₁/₂ ≈ 115 мин)
- 10⁻³ (T₁/₂ ≈ 11.5 мин)
- 10⁻² (T₁/₂ ≈ 69 сек)
- 10⁻¹ (T₁/₂ ≈ 7 сек)

**Время моделирования:** автоматически = 5 × T₁/₂ (ограничено 10-1000 сек)
